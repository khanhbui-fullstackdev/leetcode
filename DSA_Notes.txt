1. elements[:index]
Tạo một slice mới bắt đầu từ phần tử đầu tiên của mảng gốc cho đến phần tử thứ index - 1. Độ dài (Length): index
Ví dụ: a := []int{0, 1, 2, 3, 4} -> a[:3] kết quả là [0, 1, 2].

2. elements[startIndex:endIndex]
Tạo một slice mới bắt đầu từ startIndex và kết thúc tại endIndex - 1. Độ dài (Length): endIndex - startIndex
Ví dụ: a[1:4] kết quả là [1, 2, 3].

3. elements[index:]
Tạo một slice mới bắt đầu từ index cho đến hết phần tử cuối cùng của mảng gốc.
Ví dụ: a[2:] kết quả là [2, 3, 4].

------------------------------------------------------- String's theory -------------------------------------------------------
| Hàm                               | Chức năng                 | Ví dụ                                          |
| --------------------------------- | ------------------------- | ---------------------------------------------- |
| `strings.Contains(s, sub)`        | Kiểm tra substring        | `"abc", "a" → true`                            |
| `strings.HasPrefix(s, prefix)`    | Chuỗi bắt đầu bằng…       | `"hello", "he" → true`                         |
| `strings.HasSuffix(s, suffix)`    | Chuỗi kết thúc bằng…      | `"hello", "lo" → true`                         |
| `strings.Count(s, sub)`           | Đếm số lần xuất hiện      | `"banana", "a" → 3`                            |
| `strings.Index(s, sub)`           | Vị trí đầu tiên của sub   | `"go gopher", "go" → 0`                        |
| `strings.LastIndex(s, sub)`       | Vị trí cuối cùng của sub  | `"go gopher", "go" → 3`                        |
| `strings.Repeat(s, n)`            | Lặp lại chuỗi n lần       | `"a", 3 → "aaa"`                               |
| `strings.Replace(s, old, new, n)` | Thay thế chuỗi            | `"oink oink", "oink", "moo", -1` → `"moo moo"` |
| `strings.Split(s, sep)`           | Cắt chuỗi thành slice     | `"a,b,c" → []string{"a","b","c"}`              |
| `strings.Join(slice, sep)`        | Ghép slice thành chuỗi    | `["a","b"] → "a-b"`                            |
| `strings.ToLower(s)`              | Chuyển thường             | `"HELLO" → "hello"`                            |
| `strings.ToUpper(s)`              | Chuyển hoa                | `"hi" → "HI"`                                  |
| `strings.Trim(s, cutset)`         | Xóa ký tự đầu/cuối        | `"!!!hello!!!", "!" → "hello"`                 |
| `strings.TrimSpace(s)`            | Xóa khoảng trắng đầu/cuối | `"  hi  " → "hi"`                              |
| `strings.Fields(s)`               | Tách theo whitespace      | `"a b  c" → ["a","b","c"]`                     |

------------------------------------------------------ Golang vs C# String -----------------------------------------

| Purpose         | Golang (`strings` / built-in)                | C# (`string` / `System.Text`)                  | Time Complexity                                                   |
| --------------- | -------------------------------------------- | ---------------------------------------------- | ----------------------------------------------------------------- |
| ✅ Length        | `len(str)`                                   | `str.Length`                                   | **O(1)** (stored property, không phải scan chuỗi)                 |
| ✅ Indexing      | `str[i]` (byte) / `[]rune(str)[i]` (unicode) | `str[i]` (char)                                | **O(1)** (but `[]rune` conversion in Go là **O(n)** để tạo slice) |
| ✅ Substring     | `str[start:end]`                             | `str.Substring(start, len)`                    | **O(k)** (k = độ dài substring)                                   |
| ✅ Concatenation | `str1 + str2` / `strings.Builder`            | `str1 + str2` / `StringBuilder`                | `+` = **O(n+m)**, Builder amortized **O(1)** append               |
| ✅ Contains      | `strings.Contains(str, sub)`                 | `str.Contains(sub)`                            | **O(n·m)** worst-case (Boyer-Moore / IndexOf tối ưu hơn một chút) |
| ✅ Has Prefix    | `strings.HasPrefix(str, prefix)`             | `str.StartsWith(prefix)`                       | **O(m)**                                                          |
| ✅ Has Suffix    | `strings.HasSuffix(str, suffix)`             | `str.EndsWith(suffix)`                         | **O(m)**                                                          |
| ✅ Index Of      | `strings.Index(str, sub)`                    | `str.IndexOf(sub)`                             | **O(n·m)** worst-case                                             |
| ✅ Last Index Of | `strings.LastIndex(str, sub)`                | `str.LastIndexOf(sub)`                         | **O(n·m)**                                                        |
| ✅ Replace       | `strings.Replace(str, old, new, n)`          | `str.Replace(old, new)`                        | **O(n·m)** (scan toàn bộ, copy lại)                               |
| ✅ Split         | `strings.Split(str, sep)`                    | `str.Split(sep)`                               | **O(n)** (scan toàn bộ, tạo mảng mới)                             |
| ✅ Join          | `strings.Join(slice, sep)`                   | `string.Join(sep, arr)`                        | **O(n)** (copy toàn bộ + sep)                                     |
| ✅ Trim          | `strings.Trim(str, cutset)`                  | `str.Trim(chars)`                              | **O(n)**                                                          |
| ✅ Trim Left     | `strings.TrimLeft`                           | `str.TrimStart`                                | **O(n)** (worst-case)                                             |
| ✅ Trim Right    | `strings.TrimRight`                          | `str.TrimEnd`                                  | **O(n)** (worst-case)                                             |
| ✅ To Lower      | `strings.ToLower(str)`                       | `str.ToLower()`                                | **O(n)**                                                          |
| ✅ To Upper      | `strings.ToUpper(str)`                       | `str.ToUpper()`                                | **O(n)**                                                          |
| ✅ Compare       | `strings.Compare(a, b)`                      | `string.Compare(a, b)`                         | **O(min(n,m))**                                                   |
| ✅ Equals        | `a == b`                                     | `a == b`                                       | **O(n)** (worst-case; short-circuit nếu khác ngay từ đầu)         |
| ✅ Repeat        | `strings.Repeat(str, count)`                 | `string.Concat(Enumerable.Repeat(str, count))` | **O(n·count)**                                                    |



package main

import (
	"fmt"
	"strings"
	"strconv"
)

func main() {
	str := "go gopher"

	// 1. Contains / Prefix / Suffix / Count
	fmt.Println(strings.Contains(str, "go"))            // true
	fmt.Println(strings.HasPrefix(str, "go"))           // true
	fmt.Println(strings.HasSuffix(str, "her"))          // true
	fmt.Println(strings.Count("banana", "an"))          // 2

	// 2. Index / LastIndex / IndexAny
	fmt.Println(strings.Index(str, "go"))               // 0
	fmt.Println(strings.LastIndex("abracadabra", "ra")) // 9
	fmt.Println(strings.IndexAny("hello, 世界", "界!"))     // 8 (vị trí rune '界')

	// 3. Replace / ReplaceAll
	fmt.Println(strings.Replace("a-a-a", "a", "x", 2))  // x-x-a
	fmt.Println(strings.ReplaceAll("a-a-a", "a", "x"))  // x-x-x

	// 4. Split / SplitN / Join
	parts := strings.Split("a,b,c", ",")
	fmt.Println(parts)                                  // [a b c]
	fmt.Println(strings.SplitN("a,b,c", ",", 2))        // [a b,c]
	fmt.Println(strings.Join([]string{"go", "lang"}, "-")) // go-lang

	// 5. ToLower / ToUpper / EqualFold
	fmt.Println(strings.ToLower("GoLang"))              // golang
	fmt.Println(strings.ToUpper("GoLang"))              // GOLANG
	fmt.Println(strings.EqualFold("straße", "STRASSE")) // true (case-folding Unicode)

	// 6. Trim / TrimSpace / TrimPrefix / TrimSuffix
	fmt.Println(strings.Trim("  --hello--  ", " -"))    // hello
	fmt.Println(strings.TrimSpace(" \t hi \n"))         // hi
	fmt.Println(strings.TrimPrefix("foobar", "foo"))    // bar
	fmt.Println(strings.TrimSuffix("foobar", "bar"))    // foo

	// 7. Fields / FieldsFunc
	fmt.Println(strings.Fields("Go   is\tgreat\n"))     // [Go is great]
	isVowel := func(r rune) bool {
		return strings.ContainsRune("aeiouAEIOU", r)
	}
	fmt.Println(strings.FieldsFunc("h-e?l!l,o", isVowel)) // tách theo nguyên âm -> [h- -?l!l,]

	// 8. Repeat
	fmt.Println(strings.Repeat("na", 4) + " Batman!")   // nanananana Batman!

	// 9. Builder (hiệu năng tốt)
	var b strings.Builder
	b.Grow(16)
	b.WriteString("Hello")
	b.WriteByte(' ')
	b.WriteString("Gopher")
	fmt.Println(b.String())                             // Hello Gopher

	// 10. strconv Atoi / Itoa / ParseInt
	i, err := strconv.Atoi("123")
	fmt.Println(i, err)                                 // 123 <nil>
	fmt.Println(strconv.Itoa(456))                      // "456"
	// ParseInt: base 16, 32-bit
	n, _ := strconv.ParseInt("FF", 16, 32)
	fmt.Println(n)                                      // 255
}

----------------------------------------------------- Pointer in Golang -----------------------------------------
Trong Golang, toán tử & dùng để lấy ĐỊA CHỈ(Address) của chính biến đó(address of the variable/pointer itself), chứ không phải là giá trị vùng nhớ mà con trỏ đó đang trỏ tới (the memory address it points to).

headA.Next.Next là một biến con trỏ nằm trên Stack/Heap thuộc node a2. Bản thân cái con trỏ này phải nằm ở một vị trí nào đó trong bộ nhớ để lưu giá trị. Khi bạn dùng &headA.Next.Next, bạn đang lấy địa chỉ vùng nhớ của chính cái hộp lưu trữ con trỏ đó.
Tương tự, headB.Next.Next.Next là một biến con trỏ nằm ở node b3. Vì node a2 và node b3 là hai node hoàn toàn nằm ở hai vị trí khác nhau, nên địa chỉ của hai biến Next nằm bên trong chúng bắt buộc phải khác nhau (0x3b288ca9c068 và 0x3b288ca9c078 như trong hình console của bạn).

Để kiểm tra xem chúng có đang trỏ chung vào một node c1 hay không, bạn cần in ra giá trị con trỏ (địa chỉ vùng nhớ của node mục tiêu) chứ không phải địa chỉ của con trỏ.

fmt.Printf("\nPrint memory address shared node A: %p\n", headA.Next.Next)
fmt.Printf("\nPrint memory address shared node B: %p\n", headB.Next.Next.Next)
